How to run the files in the project (for ProjectCode.py):
	1. Install packages numpy, matplotlib, pandas, sklearn, and keras
	2. If running on IDE:
		1. Open the directory in a Python IDE (e.g., PyCharm or IntelliJ)
		2. Get the name of file (ProjectCode.py) and hit the run button
	3. If running on terminal:
		1. Go to directory ProjectCode.py is stored on
		2. Install packages numpy, matplotlib, pandas, and sklearn
			1. pip install numpy
			2. pip install matplotlib
			3. pip install pandas
			4. pip install sklearn
			5. pip install keras
		3. Run the file name (just type in "ProjectCode.py")
